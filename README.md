# Wanderlust

A full-stack web application built with a responsive front-end 
and RESTful back-end architecture, supporting real-time data 
rendering, user authentication, and dynamic DOM manipulation.

## Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript (DOM API), Bootstrap 5
- **Backend:** Node.js, Express.js
- **Database:** MongoDB / MySQL
- **Version Control:** Git & GitHub
